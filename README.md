# FIFA World Cup Finals Analysis (1970–2022)

## Overview
This project analyzes FIFA World Cup finals from 1970 to 2022.

## Dataset Columns
- Year
- Winner
- Runner-up
- Total Goals
- Penalty Shootout

## Charts
1. Goals in finals by year
2. Winner title count
3. Penalty shootout frequency
4. Correlation heatmap
5. Winners Distribution

## Machine Learning
Predicts possible total goals in future World Cup finals.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Run Command
pip install pandas matplotlib seaborn scikit-learn
python main.py
